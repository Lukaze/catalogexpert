Preconfigured Entitlement States Reference

🔒 InstalledAndPermanent

The app is installed and cannot be uninstalled by the user and will not show up in store. Note: This state is not to be used if the app contains bot in specified scope or resource specific permissions.

🟢 Installed

The app is installed and can be uninstalled by the user.

✅ PreConsented

The app is preconsented and gets silently installed upon first use.

⭐ Featured

The app is featured and needs to be installed by the user on first use.

❌ NotInstalled

The app is not installed but existing tabs are shown. This is also known as the grandfathered state and should not be used for any new apps.

⚠️ InstalledAndDeprecated

The app has been marked for deprecation. It will still show up in Teams Admin Center but will not show up for users in Teams Client.

🚫 HiddenFromAppStore

The app would get hidden from all app discovery entry points including Teams Client App Store and Teams Admin Center.
